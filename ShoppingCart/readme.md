# Workshop 3: ShoppingCart

This project is an implementation of a shopping cart API, developed as part of a programming workshop in my course.

## Features

1. 🛒 Add products to the shopping cart
2. ➖ Remove products from the shopping cart
3. 🔢 Update product quantities
4. 📋 View cart contents
6. ✅ Manage shopping cart operations through an API endpoints



## 🧠 What I Learned

Through this project, I gained hands-on experience with:
- How to express different relations using Entity Framework entity models. 
- API implementation using ASP.NET Core Web API.
- Using Controllers to map incoming urls to specific methods.
- Web API parameter binding 
- How to redirect to https to enforce better security.
- Using postman to test the API.
- Using swagger to document the API.   
- Controller action return types  
- HTTP response status codes
- Dependecy injection 



## Prerequisites
- .NET 10
- Postman

## Getting Started
- Clone the repository
- Run the project
  - to run "https profile" , use this command :
    - dotnet watch --launch-profile "https"
- Open Postman 
- Import API
- Use swagger json link "https://localhost:7051/swagger/v1/swagger.json"
- Adjust the base url variable value to "https://localhost:7051"
