## Second Workshop - Calculator Application - Week13: [Calculator Repo](https://github.com/T-King-00/workshop-csharp-.net/tree/master/Calculator).   

This project is a simple calculator application built as part of a C# and .NET workshop.  
Its purpose is to demonstrate core programming concepts such as basic arithmetic operations, project structure, and clean code practices in a minimal and focused way.

### Project Status:  
This project is part of an ongoing workshop and may be extended with additional features or refactoring examples.  

## Features:  

- Addition
- Subtraction
- Multiplication
- Power
- SquareRoot
- Division
- Modulus
- Console-based interaction

## 🧠 What I Used & Learned  
This project was an opportunity to practice and apply several core C# and clean code concepts.

### Clean Code Practices
- Clear and descriptive method names
- Small, focused methods with a single responsibility
- Avoiding unnecessary complexity

### C# Concepts 
- **Method Overloading**  
  Used to provide multiple ways to perform calculations while keeping method names intuitive and consistent.

- **Delegates**  
  Applied to encapsulate calculation logic and demonstrate flexible method invocation.

- **Junie (AI Agent)**  
  Used as an AI assistant to help analyze the codebase and calculate test coverage.

- **Code Coverage Analysis**  
  + Learned how to assess code coverage to better understand which parts of the code are tested and where improvements can be made.
  + Learned that code coverage shouldnot be a goal but rather a indication and metric.

### Key Takeaways
- Clean code principles improve clarity even in small projects.  
- Method overloading helps keep APIs intuitive.  
- Delegates provide a powerful way to decouple logic and behavior.
